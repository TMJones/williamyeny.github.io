Mostly projects created when I was enrolled in Suffolk County Community College. I was required by my professor to put all code in a single file, so classes had to be included in the main file. Please view my GitHub for sources and more.

Orbital

A full minigame in which you must stop the balls from exiting via an orbiting platform.

Platformer

A classic platformer in which you must reach the end without falling to your doom. Emphasis on smooth camera movement and easy map making.

Explosions

Fun physics demo that lets you see how explosions can affect a virtual ball. Bonus: particles.

Vroom

A driving demo featuring acceleration, friction and the ability to spawn unlimited "cars".

Ripple

An experiment that replicates the nature of ripples as they bounce off a wall. How does nature make it look so easy?

Physics Ball

An interesting physics demo that lets you set the angle and force to project a ball and watch how gravity and friction affects it.

Orbs and Connectors

An experiment that generates random pulsing orbs and links them with connectors.
